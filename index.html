📝 通用表单样式文档

一套支持浅色/深色双主题的现代表单样式系统，基于 SCSS 构建。默认样式直接作用于 HTML 标签，通过类名控制尺寸和变体。

💡 设计理念

本样式系统采用标签优先的设计方式：

  • 默认样式：直接作用于 HTML 标签（如 <input>、<button>），无需添加类名
  • 尺寸控制：通过 .small 和 .large 类名切换尺寸（默认中号）
  • 类型变体：通过 .primary、.danger 等类名切换样式变体
  • 状态修饰：通过 .is-error、.is-success 等类名表示状态

🚀 快速开始

在 HTML 文件的 <head> 中引入编译后的 CSS 文件：

<link rel="stylesheet" href="./css/style.css">

在 <html> 或任意父元素上添加主题类名：

<!-- 浅色主题 -->
<html class="theme-light">

<!-- 深色主题 -->
<html class="theme-dark">

使用 JavaScript 动态切换主题：

// 切换到深色主题
document.documentElement.classList.remove('theme-light');
document.documentElement.classList.add('theme-dark');

// 切换主题（toggle）
document.documentElement.classList.toggle('theme-light');
document.documentElement.classList.toggle('theme-dark');

📥 输入框 (Input)

样式直接作用于 input 标签（支持 type="text", "password", "email", "number" 等）

可用类名

类名 说明 效果
（无） 默认样式 中号尺寸，高度 32px
small 小尺寸 高度 24px，字体 12px
large 大尺寸 高度 40px，字体 16px
is-error 错误状态 红色边框，用于表单验证失败
is-success 成功状态 绿色边框，用于验证通过
is-warning 警告状态 黄色边框，用于警告提示

效果演示

使用示例

<!-- 默认中号输入框 -->
<input type="text" placeholder="请输入...">

<!-- 小号输入框 -->
<input type="text" class="small" placeholder="小号...">

<!-- 大号输入框 -->
<input type="text" class="large" placeholder="大号...">

<!-- 错误状态 -->
<input type="text" class="is-error" value="输入有误">

<!-- 成功状态 -->
<input type="text" class="is-success" value="验证通过">

<!-- 禁用状态 -->
<input type="text" disabled>

<!-- 只读状态 -->
<input type="text" readonly value="不可编辑">

📦 表单组合 (Form Group)

用于组织表单元素的组合样式

可用类名

类名 说明 效果
form-group 表单组容器 包含标签、输入框和提示信息
is-required 必填标记 在 label 前添加红色星号
form-hint 提示文字 灰色小字提示
form-error-msg 错误信息 红色错误提示文字
form-inline 行内布局 让表单组横向排列

效果演示

用户名长度为 3-20 个字符
邮箱格式不正确

使用示例

<!-- 基础表单组 -->
<div class="form-group">
  <label>用户名</label>
  <input type="text" placeholder="请输入用户名">
  <div class="form-hint">用户名长度为 3-20 个字符</div>
</div>

<!-- 必填项 + 错误提示 -->
<div class="form-group">
  <label class="is-required">邮箱</label>
  <input type="email" class="is-error" placeholder="请输入邮箱">
  <div class="form-error-msg">邮箱格式不正确</div>
</div>

<!-- 行内布局 -->
<div class="form-inline">
  <div class="form-group">
    <label>姓名</label>
    <input type="text">
  </div>
  <div class="form-group">
    <label>电话</label>
    <input type="text">
  </div>
</div>

🎨 主题系统

本样式系统内置浅色和深色两套主题，通过 CSS 变量实现无缝切换。

主题类名

类名 说明
theme-light 浅色主题（默认）
theme-dark 深色主题

主题变量

主题通过 CSS 变量定义，可在自定义样式中覆盖：

/* 表单颜色变量 */
--form-border: #d9d9d9;        /* 边框颜色 */
--form-border-hover: #40a9ff;  /* 悬停边框 */
--form-border-focus: #1890ff;  /* 聚焦边框 */
--form-bg: #ffffff;            /* 背景色 */
--form-bg-disabled: #f5f5f5;   /* 禁用背景 */
--form-text: #333333;          /* 文字颜色 */
--form-text-disabled: #bfbfbf; /* 禁用文字 */
--form-placeholder: #bfbfbf;   /* 占位符颜色 */
--form-error: #ff4d4f;         /* 错误颜色 */
--form-success: #52c41a;       /* 成功颜色 */
--form-warning: #faad14;       /* 警告颜色 */

/* 按钮颜色变量 */
--btn-primary-bg: #1890ff;     /* 主按钮背景 */
--btn-primary-bg-hover: #40a9ff; /* 主按钮悬停 */
--btn-danger-bg: #ff4d4f;      /* 危险按钮背景 */
--btn-danger-bg-hover: #ff7875; /* 危险按钮悬停 */

📁 文件结构

通用样式/
├── css/
│   └── style.css          # 编译后的样式文件（引入此文件）
├── scss/
│   ├── style.scss         # SCSS 入口文件
│   ├── theme.scss         # 主题变量定义
│   ├── form.scss          # 表单组件样式
│   └── checkbox.scss      # 复选框样式
└── index.html             # 本文档

📋 完整示例

一个包含所有表单元素的完整示例：

<!DOCTYPE html>
<html lang="zh-CN" class="theme-light">
<head>
  <meta charset="UTF-8">
  <title>表单示例</title>
  <link rel="stylesheet" href="./css/style.css">
</head>
<body>
  <form>
    <div class="form-group">
      <label class="is-required">用户名</label>
      <input type="text" placeholder="请输入用户名">
    </div>

    <div class="form-group">
      <label>邮箱</label>
      <input type="email" placeholder="请输入邮箱">
    </div>

    <div class="form-group">
      <label>性别</label>
      <label>
        <input type="radio" name="gender" checked> 男
      </label>
      <label>
        <input type="radio" name="gender"> 女
      </label>
    </div>

    <div class="form-group">
      <label>爱好</label>
      <label>
        <input type="checkbox"> 阅读
      </label>
      <label>
        <input type="checkbox"> 运动
      </label>
    </div>

    <div class="form-group">
      <label>城市</label>
      <select>
        <option>北京</option>
        <option>上海</option>
        <option>广州</option>
      </select>
    </div>

    <div class="form-group">
      <label>简介</label>
      <textarea placeholder="请输入简介..."></textarea>
    </div>

    <div class="form-group">
      <label class="switch">
        <input type="checkbox">
        <span class="slider"></span>
      </label>
      接收通知
    </div>

    <button type="submit" class="primary">提交</button>
    <button type="button" class="default">取消</button>
  </form>
</body>
</html>
