📥 输入框 (Input)

样式直接作用于 input 标签，支持多种类型和状态。

支持的类型

以下 input 类型均已应用默认样式：

  • text - 文本输入
  • password - 密码输入
  • email - 邮箱输入
  • number - 数字输入
  • search - 搜索框
  • tel - 电话输入
  • url - URL 输入
  • date - 日期选择
  • datetime-local - 日期时间选择
  • month - 月份选择
  • time - 时间选择
  • week - 周选择

可用类名

类名 说明 效果
（无） 默认样式 高度 32px
.is-error 错误状态 红色边框，用于表单验证失败
.is-success 成功状态 绿色边框，用于验证通过
.is-warning 警告状态 黄色边框，用于警告提示

使用示例

<!-- 默认中号输入框 -->
<input type="text" placeholder="请输入...">

<!-- 小号输入框 -->
<input type="text" class="small" placeholder="小号...">

<!-- 大号输入框 -->
<input type="text" class="large" placeholder="大号...">

类型演示

使用示例

<input type="text" placeholder="请输入文本">
<input type="password" placeholder="请输入密码">
<input type="email" placeholder="请输入邮箱">
<input type="number" placeholder="请输入数字">

状态演示

使用示例

<!-- 错误状态 -->
<input type="text" class="is-error" value="输入有误">

<!-- 成功状态 -->
<input type="text" class="is-success" value="验证通过">

<!-- 警告状态 -->
<input type="text" class="is-warning" value="请注意">

<!-- 禁用状态 -->
<input type="text" disabled>

<!-- 只读状态 -->
<input type="text" readonly value="不可编辑">

状态组合

💡 提示

状态类名（.is-error/.is-success/.is-warning）可以单独使用或与其他属性组合。
