☑️ 复选框 (Checkbox)

样式直接作用于 input[type="checkbox"] 标签，支持多选。

可用类名

类名 说明 效果
（无） 默认样式 尺寸 18×18px
.small 小尺寸 尺寸 14×14px
.large 大尺寸 尺寸 22×22px
.round 圆形样式 圆形复选框
.is-error 错误状态 红色边框/背景
.is-success 成功状态 绿色边框/背景
.is-warning 警告状态 黄色边框/背景

基础用法

使用示例

<label>
  <input type="checkbox"> 选项文字
</label>

<label>
  <input type="checkbox" checked> 默认选中
</label>

<!-- 禁用状态 -->
<input type="checkbox" disabled>
<input type="checkbox" disabled checked>

尺寸演示

使用示例

<!-- 小号复选框 -->
<input type="checkbox" class="small">

<!-- 默认复选框 -->
<input type="checkbox">

<!-- 大号复选框 -->
<input type="checkbox" class="large">

圆形样式

使用示例

<!-- 圆形复选框 -->
<input type="checkbox" class="round">

<!-- 圆形 + 尺寸 -->
<input type="checkbox" class="round small">
<input type="checkbox" class="round large">

状态演示

使用示例

<input type="checkbox" class="is-error">
<input type="checkbox" class="is-success">
<input type="checkbox" class="is-warning">

多选组示例

  <label><input type="checkbox" name="hobby" value="reading"> 阅读</label>

与单选框对比

特性 Checkbox Radio
选择模式 多选（可同时选多个） 单选（只能选一个）
name 属性 相同 name 也支持多选 相同 name 互斥
特殊样式 支持 .round 圆形 不支持圆形
典型应用 兴趣爱好、权限设置 性别、单选题

通用表单样式系统 · 支持浅色/深色双主题
