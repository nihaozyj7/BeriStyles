🔘 单选框 (Radio)

样式直接作用于 input[type="radio"] 标签，同一 name 的选项互斥。

可用类名

类名 说明 效果
（无） 默认样式 尺寸 16×16px
.small 小尺寸 尺寸 12×12px
.large 大尺寸 尺寸 20×20px

基础用法

请选择性别：

使用示例

<label>
  <input type="radio" name="gender" value="male" checked> 男
</label>
<label>
  <input type="radio" name="gender" value="female"> 女
</label>
<label>
  <input type="radio" name="gender" value="other"> 其他
</label>

尺寸演示

使用示例

<!-- 小号单选框 -->
<input type="radio" name="group" class="small">

<!-- 默认单选框 -->
<input type="radio" name="group">

<!-- 大号单选框 -->
<input type="radio" name="group" class="large">

状态演示

使用示例

<!-- 正常状态 -->
<input type="radio" name="group" checked>
<input type="radio" name="group">

<!-- 禁用状态 -->
<input type="radio" name="group" disabled>
<input type="radio" name="group" disabled checked>

多组单选框

性别：
学历：

使用示例

<!-- 第一组 -->
<input type="radio" name="gender" value="male"> 男
<input type="radio" name="gender" value="female"> 女

<!-- 第二组（不同的 name） -->
<input type="radio" name="education" value="high"> 高中
<input type="radio" name="education" value="college"> 大专
💡 重要提示

同一组单选框必须使用相同的 name 属性，这样才能实现互斥选择的效果。不同组的单选框应使用不同的 name。
