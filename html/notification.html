🔔 消息通知 (Notification)

全局消息通知组件，支持多种类型、位置和自定义配置，自动适配浅色/深色主题。

特性概览

  • 🎨 主题自适应 - 自动读取 CSS 变量适配浅色/深色主题
  • 📍 多位置支持 - 支持左上、右上、左下、右下四个位置
  • 🔔 多种类型 - info、success、warning、error 四种消息类型
  • ⏱️ 自动关闭 - 支持自动关闭和手动关闭
  • 🎬 动画效果 - 平滑的进入/退出动画
  • 📱 响应式设计 - 移动端适配，自动调整位置

快速开始

消息通知组件是一个纯 TypeScript 组件，无需在 HTML 中预定义结构，直接调用 API 即可显示通知。

1. 引入组件

// ES Module 方式导入
import { Notification } from '../js/notification.js';

// 或作为模块使用
import { showNotification, info, success, warning, error } from '../js/notification.js';

2. 显示通知

// 显示 info 通知
Notification.info('这是一条信息提示');

// 显示 success 通知
Notification.success('操作成功！');

// 显示 warning 通知
Notification.warning('请注意，这是一条警告');

// 显示 error 通知
Notification.error('操作失败，请重试');

自定义配置

可以通过配置对象自定义通知的各项参数：

配置示例

// 完整配置示例
Notification.show({
  title: '提示',           // 通知标题（可选）
  message: '操作成功',      // 通知内容（必填）
  type: 'success',         // 类型：info/success/warning/error
  duration: 5000,          // 显示时长（毫秒），0 表示不自动关闭
  position: 'top-right',   // 位置：top-left/top-right/bottom-left/bottom-right
  showClose: true,         // 是否显示关闭按钮
  onClose: () => {         // 关闭回调（可选）
    console.log('通知已关闭');
  }
});

API 参考

Notification.show(options)

显示通知的主要方法，接受一个配置对象。

参数 类型 必填 默认值 说明
message string 是 - 通知的文本内容
title string 否 - 通知标题
type NotificationType 否 'info' 通知类型：info/success/warning/error
duration number 否 4500 显示时长（毫秒），0 表示不自动关闭
position NotificationPosition 否 'top-right' 显示位置
showClose boolean 否 true 是否显示关闭按钮
onClose () => void 否 - 关闭时的回调函数

快捷方法

方法 参数 说明
Notification.info(message | options) string 或 NotificationOptions 显示 info 类型通知
Notification.success(message | options) string 或 NotificationOptions 显示 success 类型通知
Notification.warning(message | options) string 或 NotificationOptions 显示 warning 类型通知
Notification.error(message | options) string 或 NotificationOptions 显示 error 类型通知
Notification.closeAll() 无 关闭所有通知

类型定义

type NotificationType = 'info' | 'success' | 'warning' | 'error';
type NotificationPosition = 'top-left' | 'top-right' | 'bottom-left' | 'bottom-right';

interface NotificationOptions {
  message: string;
  title?: string;
  type?: NotificationType;
  duration?: number;
  position?: NotificationPosition;
  showClose?: boolean;
  onClose?: () => void;
}

主题适配

消息通知组件通过读取 CSS 变量自动适配当前主题，无需额外配置。

🎨 主题变量

通知组件使用以下 CSS 变量：

  • --notification-bg - 通知背景色
  • --notification-text - 通知文字颜色
  • --notification-border - 通知边框颜色
  • --notification-shadow - 通知阴影
  • --notification-info - info 类型颜色
  • --notification-success - success 类型颜色
  • --notification-warning - warning 类型颜色
  • --notification-error - error 类型颜色

自定义主题变量

如需自定义通知样式，可以覆盖以下 CSS 变量：

/* 自定义通知样式 */
.theme-light {
  --notification-bg: #ffffff;
  --notification-text: #333333;
  --notification-border: #e8e8e8;
  --notification-shadow: 0 4px 12px rgba(0, 0, 0, 0.15);
}

.theme-dark {
  --notification-bg: #1e1e1e;
  --notification-text: #e0e0e0;
  --notification-border: #434343;
  --notification-shadow: 0 4px 12px rgba(0, 0, 0, 0.5);
}

完整示例

以下是一个完整的使用示例：

// 导入通知组件
import { Notification } from './notification.js';

// 显示成功通知
Notification.success({
  title: '提交成功',
  message: '您的表单已成功提交，我们将尽快处理。',
  duration: 5000,
  position: 'top-right',
  onClose: () => {
    console.log('通知已关闭');
  }
});

// 显示错误通知（不会自动关闭）
Notification.error({
  title: '提交失败',
  message: '网络连接异常，请检查网络后重试。',
  duration: 0,  // 不自动关闭
  showClose: true
});

// 批量显示通知
function showBatchNotifications() {
  Notification.info('开始处理...');
  
  setTimeout(() => {
    Notification.success('处理完成！');
  }, 2000);
}

// 关闭所有通知
Notification.closeAll();

注意事项

⚠️ 注意

通知组件会自动创建容器元素，无需在 HTML 中预定义。确保在 DOM 加载完成后调用通知方法。

💡 提示

建议在用户操作后显示通知（如表单提交、删除确认等），避免过于频繁地显示通知影响用户体验。
