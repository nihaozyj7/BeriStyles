📋 完整示例

一个包含所有表单元素的完整注册表单示例。

效果预览

用户注册

3-20 个字符，支持字母、数字和下划线
至少 6 位字符
两次输入的密码不一致
选填，最多 200 字
接收邮件通知

源代码

<!DOCTYPE html>
<html lang="zh-CN" class="theme-light">
<head>
  <meta charset="UTF-8">
  <title>用户注册</title>
  <link rel="stylesheet" href="./css/style.css">
</head>
<body>
  <form style="max-width: 600px; margin: 0 auto; padding: 30px;">
    <h2 style="text-align: center;">用户注册</h2>

    <div class="form-group">
      <label class="is-required">用户名</label>
      <input type="text" placeholder="请输入用户名">
      <div class="form-hint">3-20 个字符</div>
    </div>

    <div class="form-group">
      <label class="is-required">邮箱</label>
      <input type="email" placeholder="请输入邮箱">
    </div>

    <div class="form-group">
      <label class="is-required">密码</label>
      <input type="password" placeholder="请输入密码">
    </div>

    <div class="form-group">
      <label class="is-required">性别</label>
      <label><input type="radio" name="gender" checked> 男</label>
      <label><input type="radio" name="gender"> 女</label>
    </div>

    <div class="form-group">
      <label>城市</label>
      <select>
        <option value="">请选择城市</option>
        <option>北京</option>
        <option>上海</option>
      </select>
    </div>

    <div class="form-group">
      <label>兴趣爱好</label>
      <label><input type="checkbox"> 阅读</label>
      <label><input type="checkbox"> 运动</label>
    </div>

    <div class="form-group">
      <label>个人简介</label>
      <textarea placeholder="请输入..."></textarea>
    </div>

    <div class="form-group">
      <label class="switch">
        <input type="checkbox" checked>
        <span class="slider"></span>
      </label>
      接收邮件通知
    </div>

    <div class="form-actions">
      <button type="submit" class="primary">注册</button>
      <button type="reset" class="default">重置</button>
    </div>
  </form>
</body>
</html>

使用说明

  1. 将上述代码保存为 HTML 文件
  2. 确保 css/style.css 文件存在（运行 pnpm run build:css 编译）
  3. 在浏览器中打开 HTML 文件即可看到效果
  4. 可以通过修改 <html> 的 class 切换主题（theme-light 或 theme-dark）
💡 提示

这个示例展示了如何组合使用各种表单组件。实际项目中，你可以根据需要选择使用部分组件，所有组件都是独立的，没有强制依赖关系。
