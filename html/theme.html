🎨 主题系统

本样式系统内置浅色和深色两套主题，通过 CSS 变量实现无缝切换。

主题类名

类名 说明
.theme-light 浅色主题（适合白天使用）
.theme-dark 深色主题（适合夜间使用）

使用方法

1. HTML 设置主题

在 <html> 或任意父元素上添加主题类名：

<!-- 浅色主题 -->
<html class="theme-light">

<!-- 深色主题 -->
<html class="theme-dark">

2. JavaScript 切换主题

// 切换到深色主题
document.documentElement.classList.remove('theme-light');
document.documentElement.classList.add('theme-dark');

// 或切换主题（toggle）
document.documentElement.classList.toggle('theme-light');
document.documentElement.classList.toggle('theme-dark');

3. 保存用户偏好

// 保存主题偏好
localStorage.setItem('theme', 'dark');

// 读取主题偏好
const savedTheme = localStorage.getItem('theme') || 'light';
document.documentElement.classList.add(`theme-${savedTheme}`);

自定义主题

1. 修改 SCSS 主题配置

编辑 scss/theme.scss 中的 $themes 映射：

$themes: (
  light: (
    bg-body: #ffffff,
    form-border: #d9d9d9,
    // ... 更多变量
  ),
  dark: (
    bg-body: #121212,
    form-border: #434343,
    // ... 更多变量
  )
);

2. 添加新主题

在 $themes 映射中添加新主题：

$themes: (
  light: ( ... ),
  dark: ( ... ),
  // 新主题
  blue: (
    bg-body: #f0f5ff,
    form-border: #adc6ff,
    // ...
  )
);

然后在 HTML 中使用：

<html class="theme-blue">

3. 运行时覆盖变量

你也可以在运行时通过 JavaScript 动态修改变量：

// 修改主题颜色
document.documentElement.style.setProperty('--form-border-focus', '#52c41a');

// 恢复默认值
document.documentElement.style.removeProperty('--form-border-focus');

主题切换最佳实践

💡 建议
  • 将用户的主题偏好保存到 localStorage 中
  • 在页面加载时立即应用保存的主题，避免闪烁
  • 考虑根据系统偏好自动设置初始主题（prefers-color-scheme）
  • 提供明显的主题切换入口，方便用户操作

根据系统偏好设置主题

// 检测系统深色模式偏好
const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
const defaultTheme = prefersDark ? 'dark' : 'light';
document.documentElement.classList.add(`theme-${defaultTheme}`);
