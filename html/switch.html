🔄 开关 (Switch)

使用 .switch 类包裹 input[type="checkbox"] 和 .slider 元素，用于表示开/关状态。

可用类名

类名 说明 效果
.switch 开关容器（必须） 默认尺寸 44×22px
.small 小尺寸开关 尺寸 32×16px
.large 大尺寸开关 尺寸 56×28px
.slider 滑块元素（必须） 开关的滑动部分

基础用法

关
开
禁用（关）
禁用（开）

使用示例

<!-- 默认开关 -->
<label class="switch">
  <input type="checkbox">
  <span class="slider"></span>
</label>

<!-- 默认开启 -->
<label class="switch">
  <input type="checkbox" checked>
  <span class="slider"></span>
</label>

<!-- 禁用状态 -->
<label class="switch">
  <input type="checkbox" disabled>
  <span class="slider"></span>
</label>

尺寸演示

小号 (32×16px)
默认 (44×22px)
大号 (56×28px)

使用示例

<!-- 小号开关 -->
<label class="switch small">
  <input type="checkbox">
  <span class="slider"></span>
</label>

<!-- 大号开关 -->
<label class="switch large">
  <input type="checkbox">
  <span class="slider"></span>
</label>

实际应用场景

接收邮件通知
自动更新
深色模式
位置服务（已禁用）

使用示例

<div class="form-group">
  <label>接收邮件通知</label>
  <label class="switch">
    <input type="checkbox" checked>
    <span class="slider"></span>
  </label>
</div>

与复选框对比

特性 Switch Checkbox
视觉样式 滑动开关 方框勾选
适用场景 设置项开关、功能启用/禁用 多选项、列表选择
交互反馈 滑动动画 勾选动画
HTML 结构 需要 label 包裹 可直接使用
💡 使用建议

开关（Switch）更适合表示功能的开启/关闭状态，通常用于设置页面；复选框（Checkbox）更适合表示选项的选择状态，通常用于表单。

通用表单样式系统 · 支持浅色/深色双主题
